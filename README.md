
# 🏠 House Price Prediction ML Project


## 📌 Description
This project predicts house prices based on:
- Area
- Bedrooms
- Bathrooms

## 🧠 ML Models Used
- Decision Tree
- Random Forest

## 🚀 Features
- Predict house price
- Deploy Web app using Streamlit
## ⚙️ Installation

#### Clone the project

```bash
  git clone https://github.com/Mudassar-23/House-Price-Prediction-App.git
```

#### Run Locally

```bash
  pip install -r requirements.txt
  python -m streamlit run app.py
```

#### Live Demo

```bash
  https://house-price-prediction-app-ml.streamlit.app/
```
