## ⚙️ Installation

#### Clone the project

```bash
  git clone https://github.com/Mudassar-23/House-Price-Prediction-App.git
```

#### Run Locally

```bash
  pip install -r requirements.txt
  python -m streamlit run app.py
```

